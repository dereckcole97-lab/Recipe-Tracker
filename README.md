# Recipe-Tracker
## Description
The recipe tracker helps to store, organize and find favourite recipes all in one place.
With the possibility to add ingredients, instructions or to categorize them by their type (breakfast, lunch, etc..) 
## Features
- Add, edit, and delete recipes
- Search for recipes by name or category
- Organize recipes by type (breakfast, brunch, lunch, snacks, dinner, dessert)
- Save ingredients and instructions
- Easy-to-use interface
## Technologies Used
- HTML
- CSS
- JavaScript
